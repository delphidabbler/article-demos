TWebBrowser Active Control Article Demo Read-me

This download contains the code that was developed in my article "How to make a TWebBrowser become the active control when clicked".

The code is targetted at Delphi 7 and has not been tested with other versions of the compiler.

There are three projects, in sub-folders 1, 2 & 3:

  1. Illustrates the problem. It is the code presented in Listings 1 and 2 in the article.
  2. Illustrates the first solution and is the code from listings 4 and 5.
  3. Illustrates the second solution and is the code from listings 6 and 7.

Load and compile each of the demo projects from their respective folders. Click in each of the three controls and see the name of the focussed control change in the status bar. Observe the different in behaviour in each of the solutions compated with the default behaviour of demo one.
