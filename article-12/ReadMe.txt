This zip archive contains the source code for a demo project that accompanies
the article "How to add a program to the Explorer Send To menu" published at
http://www.delphidabbler.com/articles.php?article=12.

USAGE

Compile the application and run it. Click the "Create Send-To Shortcut" shortcut
button to create a short cut to the program in the current user's Send To
folder.

Exit the application.

Select one or more files in Explorer and select the Send To "mouse menu". Select
the new "Send To Demo" option. The program will start and the names of the
selected file(s) will be displayed in the program.

When finished experimenting click the "Delete Send-To Shortcut" button.
