Article #10 Demo

This demo application demonstrates the use of the Internet Explorer res:// protocol. It uses the same manifest of files stored in resources discussed in the "Naming of resource" section of the DelphiDabbler article "How to create and use HTML resource files".

The page was loaded into a TWebBrowser control using the MakeResourceURL routine discussed in the article. Check out the demo's source code.
